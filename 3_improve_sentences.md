selfはインスタンスオブジェクト自身を指しています。クラスの中でインスタンスオブジェクトを呼び出す際やそのインスタンスオブジェクトを呼び出す際は、selfを使用します。
selfを実際に使ってみるためにクラスとメソッドを定義してみましょう。  
```python
>>> class Test:
...     def xxx(self):
```
Testというクラスを作り、xxxというメソッド(関数)を定義しました。ここでメソッドの引数をselfにするという決まりがあります。  
次にメソッドを呼び出してみましょう。
```python
>>> class Test:
...     def xxx(self):
...         print('abc')
... 
>>> test = Test()    #インスタンスの生成
>>> test.xxx()       #メソッドの呼び出し
abc
```
